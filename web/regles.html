Comment Jouer

A chaque tour, comparez votre carte avec celle du centre
le but est de reconnaitre les similitudes (ou non similitudes !)
entre les cartes et appuyer sur l'un des quatres boutons correspondant pour
envoyer son choix au serveur.

-Les deux : Les kirbys ont la meme forme et la meme transformation
-couleur : Les kirbys ont SEULEMENT la meme couleur
-forme : Les kirbys ont SEULEMENT la meme forme
-aucun : Les kirbys n'ont aucun point commun :'(

Systeme de points

A chaque tour, un nombre de points est attribué a chaque joueur en fonction de sa decision comme suit :

-Bonne reponse : 2 points
-Bonne reponse partielle : 1 point
-Mauvaise reponse : -1 point

Attention, si une réponse n'est pas envoyé dans le temps imparti, le score ajouté sera automatiquement de 0

1 POINT BONUS POUR LE PREMIER A AVOIR UNE REPONSE CORRECTE !
